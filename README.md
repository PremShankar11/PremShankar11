## Featured Projects

### UniEat — Student Food Marketplace
> A two-sided marketplace built for campus dining, with real-time features and AI automation.

- **Problem:** Students lacked a unified platform for ordering, splitting bills, and tracking campus food spend.
- **Solution:** Built collaborative cart syncing via Firebase and automated bill-splitting with persistent debt tracking. Integrated Gemini 2.0 Flash for restaurant onboarding — extracting menu data from physical photos with 90%+ accuracy.
- **Impact:** Reduced restaurant onboarding time by ~90%; added a real-time "Busyness" indicator to help students avoid peak-hour queues.

`React Native` `Firebase` `Gemini AI` `Razorpay`

---

### Let's Cook — Recipe Finder
> Full-stack recipe discovery app with a clean API layer and containerized deployment.

- **Problem:** No unified app for recipe search with user preference persistence and fast external data access.
- **Solution:** Built an Angular frontend backed by a Spring Boot REST API. Designed a PostgreSQL schema with external API caching for low-latency responses.
- **Impact:** Containerized the full multi-service stack with Docker Compose — reproducible and deploy-ready.

`Angular` `Spring Boot` `PostgreSQL` `Docker`

---

### ChessMate — Chess Learning App
> Java application for interactive chess learning, with an AI opponent and grandmaster match analysis.

- **Problem:** Existing chess tools lack integrated learning features for beginners and intermediate players.
- **Solution:** Built an offline-capable chess engine in Java with three AI difficulty levels using strategic move-calculation algorithms.
- **Impact:** Added grandmaster match replay and analysis tools to support structured improvement.

`Java` `AI Algorithms`
